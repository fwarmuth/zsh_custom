# robot-os plugin

The git plugin provides many [aliases](#aliases).

To use it, add `robot-os` to the plugins array in your zshrc file:

```zsh
plugins=(...robot-os)
```

## Aliases
| rcd | roscd   |
| rl  | roslaunch | 
| rs | souce ./devel/setup.sh |
| rte | rostopic echo |
| rtl | rostopic list |
